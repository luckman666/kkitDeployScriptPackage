# centos7 服务器间互信脚本


觉得不错给个star哦！！

# 具体实用步骤

git clone https://github.com/luckman666/mutual_trust.git

cd mutual_trust && chmod -R 755 .

编辑base.config参数

./mutual_trust.sh

# 脚本内容：

1、可选择是否修改yum 源

2、可选择单向或者双向互信

3、批量修改主机名

4、关闭防火墙及内核防火墙

5、服务器间时钟同步

6、可选limit句柄数优化（默认不优化）
